** Это стартовый пакет для инициализации приложения **

В пакете:

*Ангуляр (только база)

*gulp (gulp, gulp-rigger, gulp-watch)

*sass(сконфигурирована компиляция компосом)  подключены хелперы, щрифты, вариаблес бутстрапа, произведена сброска стилей


** После установки выполнить bower install и npm install
